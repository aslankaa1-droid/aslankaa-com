# aslankaa.com — сайт-визитка Аслана Кагирова

Статический сайт на 4 языках (RU / EN / FR / AR) с тремя интегрированными проектами и общей политикой.

## Структура

| Файл / папка | Назначение |
|---|---|
| `index.html` | Главная (RU) |
| `en.html`, `fr.html`, `ar.html` | Локализации |
| `policy-ru.html` … `policy-ar.html` | Соглашение об использовании сайта (4 языка) |
| `apk-orel.html`, `apk-eagles-nest-en.html` | Подпроект АПК «Орлиное гнездо» |
| `evolution/` | Подпроект «Эволюция» (ПВНК К-2) — лэндинг + RU/EN деки + PDF |
| `noah-ark/` | Подпроект «Платформа Ноев Ковчег» — лэндинг + презентация + RU/EN |
| `images/` | Растровые и векторные ассеты |
| `styles.css` | Общие стили сайта |

## Локальный просмотр

Просто открыть `index.html` двойным кликом — все ссылки относительные.
Либо запустить простой сервер:

```bash
cd "сайт визитка Аслан КАА после аудита"
python -m http.server 8000
# открыть http://localhost:8000/
```
